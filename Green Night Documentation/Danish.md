## Description
*Limit: 4000 characters*
Appen skruer kameraets eksponering og følsomhed op til maksimum for at indsamle flere fotoner. Derfor kan billedet opdatere sig langsomt. Det er ikke forsinkelse – det er fysikken bag nattesyn på en telefon.

Bildekvaliteten afhænger af din telefonmodel og mængden af omgivende lys.

Brug den udenfor om natten, hvor der er i det mindste lidt lys at forstærke. Brug den ikke i dagslys, og forvent ikke synlighed i fuldstændig mørke uden noget lys. I disse tilfælde kan skærmen se hvid ud eller billedet forblive sort.
