# Norwegian (nb)

## App Name
Green Night - Nattoptikk

## Subtitle
Nattkamera for jakt og militær

## Promotional Text
Gjør telefonen til et nattsynskamera! Se klart i mørket med grønt nattlys og IR-modus. Ideell for jakt, overvåking, militær og taktisk bruk om natten.

## Description
*Limit: 4000 characters*
Appen skrur kameraets eksponering og følsomhet opp til maksimum for å fange flere fotoner. Derfor kan bildet oppdatere seg sakte. Det er ikke forsinkelse – det er fysikken bak nattsyn på en telefon.

Bildekvaliteten avhenger av telefonmodellen din og mengden omgivende lys.

Bruk den utendørs om natten, der det finnes i det minste litt lys å forsterke. Ikke bruk den i dagslys, og ikke forvent synlighet i fullstendig mørke uten noe lys. I slike tilfeller kan skjermen bli hvit eller bildet forbli svart.
