## Description
*Limit: 4000 characters*
Appen samlar fotoner i svagt ljus, så bildhastigheten kan vara långsam. Detta är ingen fördröjning, utan nödvändigt för att kringgå kamerans begränsningar.

Bildkvaliteten beror också kraftigt på din telefonmodell och mängden omgivande ljus.

Använd inte appen på dagen (då blir skärmen vit), och i totalt mörker ser den heller ingenting. Den är optimerad för utomhusbruk på natten, precis som äkta nattkikare.
