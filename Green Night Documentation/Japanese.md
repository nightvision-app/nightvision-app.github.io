## Description
*Limit: 4000 characters*
このアプリは低照度環境で光子を蓄積するため、フレームレートが遅くなります。これはラグではなく、スマホカメラの物理的な限界を超えるために必要なプロセスです。

最終的な画質は、ご利用のスマホのモデルと周囲の光量にも大きく依存します。

日中に使用すると画面が真っ白に飛んでしまいます。また、完全に光がない密室では何も見えません。本物のナイトビジョンのように、夜間の屋外での使用に最適化されています。
