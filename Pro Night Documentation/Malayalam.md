## Description
*Limit: 4000 characters*
കുറഞ്ഞ വെളിച്ചത്തിൽ ആപ്പ് ഫോട്ടോൺ ശേഖരണം ഉപയോഗിക്കുന്നു, അതിനാൽ സെക്കൻഡിൽ ലഭിക്കുന്ന ഫ്രെയിമുകളുടെ എണ്ണം വളരെ കുറവാകാം. ഇത് ലാഗ് അല്ലെങ്കിൽ ബഗ് അല്ല, സാധാരണ ഫോണിന്റെ ക്യാമറയുടെ ഭൗതിക പരിധികളെ മറികടക്കാൻ ആവശ്യമായ പ്രക്രിയയാണ്.

അവസാന ചിത്രത്തിന്റെ ഗുണമേന്മ നിങ്ങൾ ഉപയോഗിക്കുന്ന ഫോൺ മോഡലിനും ലഭ്യമായ പരിസര വെളിച്ചത്തിന്റെ അളവിനും വളരെ ആശ്രയിച്ചിരിക്കുന്നു.

ആപ്പ് ശരിയായി പ്രവർത്തിക്കാൻ കുറഞ്ഞത് അല്പം വെളിച്ചമെങ്കിലും വേണം, ഉദാഹരണത്തിന് പുറത്തുള്ള നക്ഷത്രങ്ങൾ നിറഞ്ഞ ആകാശമോ ചന്ദ്രപ്രകാശമോ. ജനലുകളില്ലാത്ത അടച്ച മുറിപോലെയുള്ള പൂർണ്ണ ഇരുട്ടിൽ ആപ്പിന് ചിത്രം സൃഷ്ടിക്കാൻ കഴിയില്ല. അതേസമയം, പകൽ വെളിച്ചത്തിലോ ശക്തമായ വെളിച്ചത്തിലോ സെൻസറിലേക്ക് അത്യധികം വെളിച്ചം എത്തുന്നതിനാൽ സ്ക്രീൻ പൂർണ്ണമായും വെള്ളയായി മാറും.
