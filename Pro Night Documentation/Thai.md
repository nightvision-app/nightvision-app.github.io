## Description
*Limit: 4000 characters*
แอปนี้ใช้การสะสมโฟตอนในสภาพแสงน้อย ดังนั้นจำนวนเฟรมต่อวินาทีอาจค่อนข้างต่ำ นี่ไม่ใช่อาการกระตุกหรือบั๊ก แต่เป็นกระบวนการที่จำเป็นซึ่งช่วยชดเชยข้อจำกัดทางกายภาพของกล้องโทรศัพท์ทั่วไป

คุณภาพของภาพสุดท้ายยังขึ้นอยู่กับรุ่นโทรศัพท์ของคุณและปริมาณแสงแวดล้อมที่มีอยู่ด้วย

เพื่อให้แอปทำงานได้อย่างถูกต้อง จำเป็นต้องมีแหล่งกำเนิดแสงอย่างน้อยเล็กน้อย เช่น ท้องฟ้าที่มีดาวหรือแสงจันทร์กลางแจ้ง ในความมืดสนิท เช่น ห้องปิดที่ไม่มีหน้าต่าง แอปจะไม่สามารถสร้างภาพได้ ในทางกลับกัน หากอยู่ในแสงกลางวันหรือแสงจ้า หน้าจอจะกลายเป็นสีขาวทั้งหมด เพราะเซนเซอร์รับแสงมากเกินไป
