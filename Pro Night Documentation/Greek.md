## Description
*Limit: 4000 characters*
Η εφαρμογή αξιοποιεί τη συσσώρευση φωτονίων σε συνθήκες χαμηλού φωτισμού, επομένως τα καρέ ανά δευτερόλεπτο μπορεί να είναι αρκετά λίγα. Αυτό δεν είναι κόλλημα ούτε σφάλμα, αλλά μια απαραίτητη διαδικασία που παρακάμπτει τα φυσικά όρια της κάμερας ενός συνηθισμένου κινητού.

Η τελική ποιότητα εικόνας εξαρτάται επίσης σε μεγάλο βαθμό από το μοντέλο κινητού που χρησιμοποιείτε και την ποσότητα του διαθέσιμου περιβάλλοντος φωτός.

Για να λειτουργήσει σωστά η εφαρμογή, χρειάζεται έστω μια μικρή πηγή φωτός, όπως ένας έναστρος ουρανός ή το φως του φεγγαριού σε εξωτερικό χώρο. Σε απόλυτο σκοτάδι, για παράδειγμα σε κλειστό δωμάτιο χωρίς παράθυρα, η εφαρμογή δεν θα μπορέσει να δημιουργήσει εικόνα. Αντίθετα, στο φως της ημέρας ή σε έντονο φως, η οθόνη θα γίνει εντελώς λευκή επειδή ο αισθητήρας θα υπερκορεστεί.
