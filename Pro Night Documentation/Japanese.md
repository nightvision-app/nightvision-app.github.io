## Description
*Limit: 4000 characters*
このアプリは低照度環境で光子を蓄積する仕組みを使うため、1秒あたりのフレーム数がかなり少なくなることがあります。これはラグや不具合ではなく、一般的なスマホカメラの物理的な限界を補うために必要な処理です。

最終的な画質は、お使いのスマホの機種や周囲光の量にも大きく左右されます。

アプリが正しく動作するには、屋外の星空や月明かりなど、最低限の光源が必要です。窓のない閉め切った部屋のような完全な暗闇では、アプリは画像を作成できません。逆に、昼間や強い光の下では、センサーが飽和して画面が完全に白くなります。
