## Description
*Limit: 4000 characters*
本应用在弱光环境下通过光子累积工作，因此每秒帧数可能较低。这不是卡顿或故障，而是突破普通智能手机摄像头的物理限制所需的过程。

最终图像质量也很大程度上取决于你的具体手机型号以及可用环境光的多少。

为了让应用正常工作，至少需要一点微弱光源，例如户外的星空或月光。在完全黑暗的环境中，例如没有窗户的封闭房间，应用将无法形成图像。相反，在白天或强光下使用时，屏幕会因为传感器过度饱和而变成全白。
